# Refactoring
Refactoring-JavaScript

## 시작 가이드
### Requirements
For building and running the application you need:
- [Node.js 17.0.0](https://nodejs.org/ca/blog/release/v17.0.0)
- nodemon 3.0.1

### Installation
```shell
$ npm install
$ node ch6/6-1.js
```

- 자동 실행
```shell
$ npm install -g nodemon
$ nodemon ch6/6-1.js
```
